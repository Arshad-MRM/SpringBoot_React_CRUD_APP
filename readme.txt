SpringBoot Backend Server is running on Port: 9191

MySql server is running on Port: 3307 (Not default port 3306)
(If you want to change this port, you should change this in  application.properties in backend)

React Frontend is running on Port: 3000 (default)
(If you want to change this port, you want to change the same as port in Backend in Controller Anotation and, 
main Application in Backend anotation)
